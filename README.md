# RA Motors Prospect

Sistema de prospeccao automatizada de veiculos via WhatsApp + IA.

## Stack
- Backend: FastAPI + PostgreSQL (Neon) + Redis (Upstash)
- Worker: Celery + Playwright + Evolution API
- IA: Groq (Llama 3.3 70B)
- Hospedagem: Render (backend) + Maquina local (worker + scraper)
